# Nova MP Backend

Backend API con Express, TypeScript, Nodemon, Morgan y Colors.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Compilar

```bash
npm run build
```

## Producción

```bash
npm start
```

## Rutas

- `GET /` - API status
- `GET /api/health` - Health check
